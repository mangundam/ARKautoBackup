0.This program is made for myself to host server by ARK Server Manager, And it only auto backup TheIsland, because i'm tired to type map name ecah time open it, if you want to host another maps just post issues. If you use another way to host server that will be something diffrent.

1.Put ARKautoBackup.exe to the server's save file's path, server file name would be mapname.ark (if server map is "TheIsland", the save file will named "TheIsland.ark").In ARK server manager, path will be "X:\ARK Server Manager\Servers\ShooterGame\Saved\SavedArks".

2.You can open this program in anytime, this program will realtime detect if ark file is changed it will zip them into a backup file. In ARK Server Manager's setting there is a scroll bar (or scale bar?) call "Auto Svae Period", this bar is mean how often the server write file from memory. And my program will auto backup when the "TheIsland.ark" has been rewrite.

3.backup file will save in Backups and sort by each days, if you want to recovery save file, just unzip it to TheIsland.ark's path.

0.這個程式是我做來給自己透過ARK伺服器管理器開的伺服器用的，這個版本他只適用於島嶼地圖，因為如果手動輸入的話每次都要重打很麻煩，有需要其他地圖的自動備份只要在issues上貼文我可以上傳其他版本，如果不是用伺服器管理器的話使用起來效果會不一樣。

1.把ARKautoBackup.exe程式丟到伺服器存檔的地方，存檔檔名叫做"TheIsland.ark"，在伺服器管理器裡面的路徑是"X:\ARK Server Manager\Servers\ShooterGame\Saved\SavedArks"。

2.隨時都可以開啟程式，他會即時偵測存檔的檔案變更並且存成壓縮檔，在伺服器管理器裡面有一個選項叫"自動保存時間"，他的意思是隔多久會存記憶體裡面把資料存到檔案堶情A而每次寫入檔案就會變更，我的程式就會自動進行備份。

3.備份檔案會在Backups的資料夾裡面，每天再分一個子資料夾，要復原紀錄只需要把壓縮裡面的檔案覆蓋到存檔路徑上就可以了。
